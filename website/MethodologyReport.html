Golf Performance Analysis Methodology

Purpose: To evaluate player performance relative to their handicap on a hole-by-hole basis.

The Algorithm

1. Calculate Course Handicap (CH)

Determine strokes received based on tees.

CH = (HI * (Slope / 113)) + (Rating - Par)

2. Allocate Strokes (Net Score)

Distribute CH strokes based on Stroke Index (SI).

  • Base: CH // 18 strokes per hole.
  • Extra: Remaining strokes go to hardest holes (lowest SI).
  • Net Score = Gross - Strokes Received

3. Performance Metrics (Net Differential)

We compare Avg Net Score to Par.

< -0.5 Major Strength
-0.5 to +0.5 Neutral / Expected
> +0.5 Weakness / Leak
